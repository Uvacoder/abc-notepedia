
# Lists

|                                         |                                           |
| --------------------------------------- | ----------------------------------------- |
| `<li>`                                  | List Item                                 |
| `<ul> </ul>`                            | Unordered List                            |
| `<ul style="list-style-type:*circle*">` | disk *(default)* / circle / square / none |
| `<ol type=? start=?> </ol>`             | Ordered List                              |
| type="1"                                | 1. 2. 3.                                  |
| type="A"                                | A. B. C.                                  |
| type="a"                                | a. b. c.                                  |
| type="I"                                | I. II. III.                               |
| type="i"                                | i. ii. iii.                               |
